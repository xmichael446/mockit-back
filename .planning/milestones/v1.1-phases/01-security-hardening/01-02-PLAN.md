```python
REST_FRAMEWORK = {
    "DEFAULT_AUTHENTICATION_CLASSES": [
        "rest_framework.authentication.TokenAuthentication",
    ],
    "DEFAULT_PERMISSION_CLASSES": [
        "main.permissions.IsEmailVerified",
    ],
    "DEFAULT_THROTTLE_CLASSES": [
        "rest_framework.throttling.ScopedRateThrottle",
    ],
    "DEFAULT_THROTTLE_RATES": {
        "register": "10/hour",
        "guest_join": "20/hour",
        "accept_invite": "20/hour",
    },
}
```

This uses ScopedRateThrottle as a default class. Views that do NOT set `throttle_scope` are unaffected (ScopedRateThrottle skips views without a scope). The three scopes have differentiated rates per the CONTEXT.md decision: register is 10/hour (one-shot action), guest_join and accept_invite are 20/hour (headroom for shared IPs in classrooms).
