```python

# ─── Logging ──────────────────────────────────────────────────────────────────

LOGGING = {
    "version": 1,
    "disable_existing_loggers": False,
    "formatters": {
        "audit": {
            "format": "[AUDIT] %(message)s",
        },
        "standard": {
            "format": "%(asctime)s %(levelname)s %(name)s: %(message)s",
        },
    },
    "handlers": {
        "console": {
            "class": "logging.StreamHandler",
            "formatter": "standard",
        },
        "audit_file": {
            "class": "logging.FileHandler",
            "filename": BASE_DIR / "logs" / "audit.log",
            "formatter": "audit",
        },
        "audit_console": {
            "class": "logging.StreamHandler",
            "formatter": "audit",
        },
    },
    "loggers": {
        "mockit.audit": {
            "handlers": ["audit_file", "audit_console"],
            "level": "INFO",
            "propagate": False,
        },
        "mockit.email": {
            "handlers": ["console"],
            "level": "WARNING",
            "propagate": False,
        },
    },
}
```

This configures:
- `mockit.audit` logger: outputs to both `logs/audit.log` file and console with `[AUDIT]` prefix
- `mockit.email` logger: outputs to console at WARNING level (used by Plan 01's email error handling)
