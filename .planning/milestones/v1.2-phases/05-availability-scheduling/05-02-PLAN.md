class AvailabilitySlot(TimestampedModel):
    class DayOfWeek(models.IntegerChoices):
        MON = 0, "Monday"
        TUE = 1, "Tuesday"
        # ... through SUN = 6
    examiner = models.ForeignKey(settings.AUTH_USER_MODEL, on_delete=CASCADE, related_name="availability_slots")
    day_of_week = models.PositiveSmallIntegerField(choices=DayOfWeek.choices)
    start_time = models.TimeField()
    class Meta:
        unique_together = [("examiner", "day_of_week", "start_time")]

class BlockedDate(TimestampedModel):
    examiner = models.ForeignKey(settings.AUTH_USER_MODEL, on_delete=CASCADE, related_name="blocked_dates")
    date = models.DateField()
    reason = models.CharField(max_length=255, blank=True)
    class Meta:
        unique_together = [("examiner", "date")]

2. Create `scheduling/views.py` with these view classes (all DRF APIView, following project pattern):

   Define helper at top: `def _is_examiner(user): return user.role == User.Role.EXAMINER`

   **AvailabilitySlotListCreateView(APIView):**
   - `GET`: if not examiner -> 403 "Only examiners can view availability slots.". Filter `AvailabilitySlot.objects.filter(examiner=request.user)`, return serialized list.
   - `POST`: if not examiner -> 403 "Only examiners can create availability slots.". Validate with serializer, `serializer.save(examiner=request.user)`, return 201.

   **AvailabilitySlotDetailView(APIView):**
   - `patch(self, request, pk)`: if not examiner -> 403. Fetch with `get_object_or_404(AvailabilitySlot, pk=pk, examiner=request.user)` (ownership check baked in per Pitfall 3). Validate partial=True, save, return 200.
   - `delete(self, request, pk)`: if not examiner -> 403. Fetch with `get_object_or_404(AvailabilitySlot, pk=pk, examiner=request.user)`. Delete, return 204 empty.

   **BlockedDateListCreateView(APIView):**
   - `GET`: if not examiner -> 403. Filter `BlockedDate.objects.filter(examiner=request.user)`, return serialized list.
   - `POST`: if not examiner -> 403. Validate, `serializer.save(examiner=request.user)`, return 201.

   **BlockedDateDetailView(APIView):**
   - `delete(self, request, pk)`: if not examiner -> 403. Fetch with `get_object_or_404(BlockedDate, pk=pk, examiner=request.user)`. Delete, return 204.

   **ExaminerAvailableSlotsView(APIView):**
   - `GET /api/scheduling/examiners/<pk>/available-slots/?week=YYYY-MM-DD&timezone=TZ`:
   - Verify examiner exists: `get_object_or_404(User, pk=pk, role=User.Role.EXAMINER)`
   - Parse `week` query param: required, wrap `date.fromisoformat(week_param)` in try/except ValueError -> 400 "Invalid week format. Use YYYY-MM-DD."
   - Call `compute_available_slots(pk, week_date)`
   - If `timezone` query param provided, convert start/end times using `zoneinfo.ZoneInfo`. Wrap in try/except `(ZoneInfoNotFoundError, KeyError)` -> fall back to UTC.
   - Return `Response(result)`

   **ExaminerIsAvailableView(APIView):**
   - `GET /api/scheduling/examiners/<pk>/is-available/`:
   - Verify examiner exists: `get_object_or_404(User, pk=pk, role=User.Role.EXAMINER)`
   - Call `is_currently_available(pk)`
   - Return `Response(result)`
