   - Auth: Token (examiner or candidate, any session participant)
   - Response 200 when DONE: Full JSON with job_id, status, transcript, error_message, scores array

   - Status values: "Pending", "Processing", "Done", "Failed"
   - Error table:
     | 401 | Authentication credentials were not provided. |
     | 403 | You are not a participant of this session. |
     | 404 | Not found. / No AI feedback job found for this session. |

4. Notes section mentioning:
   - Retrying after FAILED is allowed (creates a new job)
   - Monthly limit default is 10 per examiner; FAILED jobs do not count toward the limit
