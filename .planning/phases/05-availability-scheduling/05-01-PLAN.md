3. Implement `scheduling/services/availability.py` (GREEN phase):

   **compute_available_slots(examiner_id: int, week_date: date) -> list[dict]:**
   - Compute `week_start = week_date - timedelta(days=week_date.weekday())` (Monday)
   - `week_end = week_start + timedelta(days=7)`
   - Fetch all `AvailabilitySlot.objects.filter(examiner_id=examiner_id)`
   - Fetch booked sessions: `IELTSMockSession.objects.filter(examiner_id=examiner_id, status__in=[SessionStatus.SCHEDULED, SessionStatus.IN_PROGRESS], scheduled_at__isnull=False, scheduled_at__date__gte=week_start, scheduled_at__date__lt=week_end).values_list("scheduled_at", flat=True)` — note the `scheduled_at__isnull=False` filter per Pitfall 6
   - Convert to `booked_starts = set(booked_sessions)`
   - Fetch blocked dates: `BlockedDate.objects.filter(examiner_id=examiner_id, date__gte=week_start, date__lt=week_end).values_list("date", flat=True)` -> `blocked_dates = set(...)`
   - Loop 7 days (offset 0-6), for each day filter slots by `day_of_week == current_day.weekday()`, expand each to `slot_start_dt = datetime.combine(current_day, slot.start_time, tzinfo=dt_timezone.utc)` and `slot_end_dt = slot_start_dt + timedelta(hours=1)`
   - Status logic: if `current_day in blocked_dates` -> "blocked", elif `any(slot_start_dt <= bs < slot_end_dt for bs in booked_starts)` -> "booked", else -> "available"
   - Each slot dict: `{"slot_id": slot.id, "start": slot.start_time.strftime("%H:%M"), "end": slot_end_dt.strftime("%H:%M"), "status": status}`
   - Sort day_slots by start, append `{"date": current_day.isoformat(), "slots": day_slots}`

   **is_currently_available(examiner_id: int) -> dict:**
   - `now = timezone.now()` (UTC-aware)
   - `today = now.date()`, `current_time = now.time()`
   - Check `BlockedDate.objects.filter(examiner_id=examiner_id, date=today).exists()` -> if True, return `{"is_available": False, "current_slot": None}`
   - Loop `AvailabilitySlot.objects.filter(examiner_id=examiner_id, day_of_week=today.weekday())`, compute `slot_end = (datetime.combine(today, slot.start_time) + timedelta(hours=1)).time()`, check `slot.start_time <= current_time < slot_end`
   - If no matching slot: return `{"is_available": False, "current_slot": None}`
   - Check for booked session: `slot_start_dt = datetime.combine(today, matching_slot.start_time, tzinfo=dt_timezone.utc)`, `slot_end_dt = slot_start_dt + timedelta(hours=1)`, query `IELTSMockSession.objects.filter(examiner_id=examiner_id, status__in=[SessionStatus.SCHEDULED, SessionStatus.IN_PROGRESS], scheduled_at__gte=slot_start_dt, scheduled_at__lt=slot_end_dt).exists()`
   - If booked: return `{"is_available": False, "current_slot": {"start": matching_slot.start_time.strftime("%H:%M"), "day_of_week": matching_slot.day_of_week}}`
   - Else: return `{"is_available": True, "current_slot": {"start": matching_slot.start_time.strftime("%H:%M"), "day_of_week": matching_slot.day_of_week}}`
