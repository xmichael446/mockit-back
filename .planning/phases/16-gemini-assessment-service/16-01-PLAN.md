    Steps:
    a. **Get audio path**: `recording = job.session.recording` then `audio_path = recording.audio_file.path`. Check `os.path.isfile(audio_path)` — raise `RuntimeError(f"Audio file not found: {audio_path}")` if missing.

    c. **Build user prompt**: Combine questions and instruction to assess:
       ```python

           f"Session questions asked:\n{questions_text}\n\n"
           "Assess the candidate's IELTS Speaking performance from the audio above. "
           "Produce a verbatim transcript and band scores with feedback for all four criteria."
       )
       ```

    d. **Create Gemini client**: `client = genai.Client(api_key=settings.GEMINI_API_KEY)`

    e. **Upload audio**: `uploaded_file = client.files.upload(file=audio_path, config=types.UploadFileConfig(mime_type="audio/webm"))` then `time.sleep(2)` for processing.

    f. **Call generate_content with retry**:
       ```python
       config = types.GenerateContentConfig(
           system_instruction=SYSTEM_PROMPT,
           response_mime_type="application/json",
           response_schema=IELTSAssessment,
       )
       max_retries = 3
       for attempt in range(1, max_retries + 1):
           try:
               response = client.models.generate_content(
                   model="gemini-2.5-pro",
                   contents=[uploaded_file, user_prompt],
                   config=config,
               )
               break
           except Exception as exc:
               status = getattr(exc, "status_code", None)
               if status in (503, 429) and attempt < max_retries:
                   time.sleep(5 * attempt)
               else:
                   raise
       ```

    g. **Safety filter check**:
       ```python
       candidate = response.candidates[0]
       if candidate.finish_reason != types.FinishReason.STOP:
           raise RuntimeError(
               f"Gemini content blocked: finish_reason={candidate.finish_reason}"
           )
       ```

    h. **Parse response**:
       ```python
       try:
           parsed = IELTSAssessment.model_validate_json(response.text)
       except Exception as exc:
           raise RuntimeError(f"Failed to parse Gemini response: {exc}") from exc
       ```

    i. **Build scores list**:
       ```python
       scores = [
           {"criterion": CRITERION_MAP[key], "band": getattr(parsed, key).band, "feedback": getattr(parsed, key).feedback}
           for key in CRITERION_MAP
       ]
       transcript = parsed.transcript
       ```

    j. **Log and return**: `return scores, transcript`
