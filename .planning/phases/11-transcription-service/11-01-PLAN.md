  <action>
Create `session/services/transcription.py` with a single function `transcribe_session(job)` that:

1. Imports `WhisperModel` from `faster_whisper` inside the function (lazy/deferred import). If ImportError, raise `RuntimeError("faster-whisper is not installed. Run: pip install faster-whisper")`.

2. Validates the recording exists:
   - Access `job.session.recording` in a try/except. If `RelatedObjectDoesNotExist` or any exception, raise `RuntimeError("Session has no associated recording.")`.
   - Get `audio_path = recording.audio_file.path`. If falsy, raise `RuntimeError("SessionRecording.audio_file is empty.")`.
   - Check file exists on disk with `os.path.isfile(audio_path)`. If not, raise `RuntimeError(f"Audio file not found: {audio_path}")`.

3. Builds the `initial_prompt` string from all SessionQuestions for the session:
   ```python
   from session.models import SessionQuestion
   questions = (
       SessionQuestion.objects
       .filter(session_part__session=job.session)
       .select_related("question")
       .order_by("session_part__part", "order")
   )
   prompt_parts = [sq.question.text for sq in questions if sq.question.text]
   initial_prompt = ". ".join(prompt_parts) if prompt_parts else None
   ```

4. Loads the WhisperModel lazily:
   ```python
   model_size = getattr(settings, "WHISPER_MODEL_SIZE", "base")
   model = WhisperModel(model_size, device="cpu", compute_type="int8")
   ```

5. Calls `model.transcribe(audio_path, beam_size=5, initial_prompt=initial_prompt, language="en")`.

6. Consumes the segments generator in a SINGLE PASS (do not iterate twice — the generator exhausts):
   ```python
   lines = []
   for segment in segments:
       lines.append(segment.text.strip())
   return "\n".join(lines)
   ```
   Note: Use plain text without speaker labels for now (no diarization available). This is simpler and Phase 12 Claude API can interpret context.

7. Add a module-level `logger = logging.getLogger(__name__)` and log the model size used and number of segments transcribed.

The function signature: `def transcribe_session(job) -> str:` — returns the transcript string. Raises RuntimeError on any unrecoverable error. The caller (run_ai_feedback task) handles status updates.
